# Youtube to Genius lyrics - Browser extension

A browser extension for searching the lyrics of the music video that you are watching on youtube.

## Tech Stack
- Monorepo
- pnpm

### Front-end - Browser extension
- ReScript
- React
- Axios
- Tailwind
- CRA
- Jest
- React testing library

### Back-end - Rest API
- Nodejs
- Typescript
- Fastify
- Genius API
- SWC
- Jest
